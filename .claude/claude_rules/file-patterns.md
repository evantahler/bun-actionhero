# File Patterns

File pattern configurations for formatting and linting.

## TypeScript Files
- Pattern: `*.ts`
- Formatter: Prettier
- Linter: ESLint

## TypeScript React Files
- Pattern: `*.tsx`
- Formatter: Prettier
- Linter: ESLint

## JavaScript Files
- Pattern: `*.js`
- Formatter: Prettier
- Linter: ESLint

## JavaScript React Files
- Pattern: `*.jsx`
- Formatter: Prettier
- Linter: ESLint

## JSON Files
- Pattern: `*.json`
- Formatter: Prettier

## Markdown Files
- Pattern: `*.md`
- Formatter: Prettier
